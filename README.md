## 👋 À propos

Bienvenue sur le dépôt de mon portfolio interactif. Je m'appelle **Victor Gomes Silva** et je suis actuellement en deuxième année de BTS SIO (Solutions Logicielles et Applications Métiers) au Lycée Léonard de Vinci à Melun. 

Ce site a été conçu de A à Z pour présenter mon parcours, mes compétences techniques et mes réalisations professionnelles.

## ✨ Fonctionnalités du site

* 🌑 **Design immersif** : Interface sombre type "terminal" avec animations au défilement.
* 📊 **HUD Dynamique** : Panneaux latéraux simulant un monitoring système (VRAM, GPU, RAM) en temps réel avec un flux de code.
* 💻 **Terminal interactif** : Animation simulant des commandes de console (`whoami`, `cat formation.txt`).
* 🪟 **Modales fluides** : Présentation détaillée des projets et rapports de stage sans recharger la page.
